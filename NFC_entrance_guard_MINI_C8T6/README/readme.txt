/****************关于工程结构****************************/
*
*所有的功能性API函数最终都会在工程文件夹app/main.c得到最后调用。
*可以通过这里开始往回追溯读者感兴趣的功能模块


*该工程的一共涉及到的底层功能大致有6个：

*串口  延时   PWM    oled&iic   sys(Interrupt priority configuration)   IWDG

*每种功能都在工程文件夹bsp/有对应文件。

*工程文件夹fwlib/是标准库文件夹，即以封装好的基本支持文件

*工程文件startup/是启动文件夹，内部包含内核文件，和与芯片对应的启动文件，表述了代码的起始地址等等信息。
*********************************************************/
